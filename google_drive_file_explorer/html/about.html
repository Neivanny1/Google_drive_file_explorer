About Us

The primary aim of this project was to develop an Intelligent File Organization and Storage System on Google Cloud Platform (Google Drive) to enhance the efficiency and effectiveness of file storage and retrieval. The significance of the project was rooted in the need to optimize the use of cloud storage systems for businesses and individuals by leveraging intelligent algorithms and techniques to automate file organization and retrieval processes.

What We Do

We try to build an intelligent file organisation System based on google cloud platform which allows users upload, download sort files in intelligent manner using automated tasks making file handling simpler and efficient.
